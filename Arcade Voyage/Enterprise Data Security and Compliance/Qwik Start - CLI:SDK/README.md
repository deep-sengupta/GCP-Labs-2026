# Cloud Storage: Qwik Start - CLI/SDK

<h2>📋 Steps</h2>

<ol>

<li>Open <b>Cloud Shell</b>.</li>

<li>Run the following commands:</li>

```bash
curl -LO http://raw.githubusercontent.com/deep-sengupta/GCP-Labs-2026/refs/heads/master/Arcade%20Voyage/Enterprise%20Data%20Security%20and%20Compliance/Qwik%20Start%20-%20CLI%3ASDK/GSP074.sh
sudo chmod +x GSP074.sh

```

<li>Copy only the <b>Region</b> value provided in the lab instructions.</li>

<li>When the script prompts for the region, paste the copied region name and press <b>Enter</b>.</li>
