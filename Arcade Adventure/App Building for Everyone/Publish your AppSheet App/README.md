# Publish your AppSheet App

<h2>📋 Steps</h2>

<ol>
  <li>Open the provided <b>Customer Contacts</b> template link.</li>

  <li>Click the <b>📄 Copy App</b> icon from the left menu.</li>

  <li>
    Configure the application:
    <ul>
      <li><b>App Name:</b> Customer Contacts</li>
      <li>Leave all remaining settings as their default values.</li>
    </ul>
  </li>

  <li>Click <b>Copy app</b>.</li>

  <li>Wait for AppSheet to finish creating the application.</li>

  <li>Open the spreadsheet and select the <b>ShipTo</b> sheet.</li>

  <li>Click <b>File → Import</b>.</li>

  <li>Upload the <code>GSP1030.xlsx</code> file.</li>

  <li>
    Configure the import settings:
    <ul>
      <li>Select <b>Replace spreadsheet</b>.</li>
      <li>Click <b>Import data</b>.</li>
    </ul>
  </li>
</ol>
