# App Building with AppSheet: Challenge Lab

<h2>📋 Steps</h2>

<ol>
  <li>Download the provided <b>contacts.xlsx</b> file.</li>

  <li>Open the provided <b>companies</b> Google Sheet.</li>

  <li>
    Import the data:
    <ul>
      <li>Click <b>File → Import</b>.</li>
      <li>Select the downloaded <b>contacts.xlsx</b> file.</li>
      <li>Choose <b>Replace spreadsheet</b>.</li>
      <li>Click <b>Import data</b>.</li>
    </ul>
  </li>

  <li>Open the <b>AppSheet</b> dashboard.</li>

  <li>Click <b>Create → App → Start with existing data</b>.</li>

  <li>
    Configure the application:
    <ul>
      <li><b>App Name:</b> Company Details</li>
      <li><b>Category:</b> Customer Engagement</li>
    </ul>
  </li>

  <li>
    Connect the data source:
    <ul>
      <li>Click <b>Choose your data</b>.</li>
      <li>Select <b>Google Sheets</b>.</li>
      <li>Select the <b>companies.xlsx</b> spreadsheet.</li>
    </ul>
  </li>

</ol>
