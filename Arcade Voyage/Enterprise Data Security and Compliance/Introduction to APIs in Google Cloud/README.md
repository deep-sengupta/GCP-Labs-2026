# Introduction to APIs in Google Cloud

<h2>📋 Steps</h2>

<ol>

<li>Open <b>Cloud Shell</b>.</li>

<li>Run the following commands:</li>

```bash
curl -LO <SCRIPT_URL>
sudo chmod +x GSP294.sh
./GSP294.sh
```

<li>Copy the <b>Region</b> value provided in the lab instructions.</li>

<li>When the script prompts for the region, paste the copied region name and press <b>Enter</b>.</li>

<li>Open the <b>OAuth 2.0 Playground</b>.</li>

<li>Search for <b>Cloud Storage API v1</b> and select the first result.</li>

<li>Click <b>Authorize APIs</b>.</li>

<li>Select your Google account and grant the requested permissions.</li>

<li>Click <b>Exchange authorization code for tokens</b>.</li>

<li>Copy the generated <b>Access Token</b>.</li>

<li>Return to <b>Cloud Shell</b>, paste the access token when prompted, and press <b>Enter</b>.</li>

</ol>
