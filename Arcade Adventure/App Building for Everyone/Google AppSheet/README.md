# Google AppSheet: Getting Started

<h2>📋 Steps</h2>

<ol>
  <li>Open the <b>AppSheet</b> dashboard.</li>

  <li>
    Click <b>Create → App → Start with existing data</b>.
  </li>

  <li>
    Configure the application:
    <ul>
      <li><b>App Name:</b> Customer Contacts</li>
      <li><b>Category:</b> Customer Engagement</li>
    </ul>
  </li>

  <li>
    Connect the data source:
    <ul>
      <li>Click <b>Choose your data</b>.</li>
      <li>Select <b>Google Sheets</b>.</li>
      <li>Select the provided <b>contacts</b> spreadsheet from Google Drive.</li>
      <li>Click <b>Customize with AppSheet</b>.</li>
    </ul>
  </li>

  <li>Wait for AppSheet to automatically generate the application.</li>

  <li>
    Preview the application:
    <ul>
      <li>Browse the list of contacts.</li>
      <li>Select any contact to view its detailed information.</li>
      <li>Verify that the data is loaded from the connected Google Sheet.</li>
    </ul>
  </li>

  <li>
    Edit a contact:
    <ul>
      <li>Click the <b>✏️ Edit</b> button.</li>
      <li>Update the contact's first name and phone number.</li>
      <li>Click <b>Save</b>.</li>
      <li>Confirm the updated information appears in both the contact details and contact list.</li>
    </ul>
  </li>

  <li>
    Add a new contact:
    <ul>
      <li>Click the <b>➕</b> button.</li>
      <li>Enter the following details:</li>
    </ul>

| Field | Value |
| :---- | :---- |
| **ID** | `51` |
| **Email Address** | `test.contact@gmail.com` |
| **First Name** | `Test` |
| **Last Name** | `Contact` |
| **Phone** | `1-100-234567` |

  </li>

  <li>
    Click <b>Save</b> to create the contact.
  </li>

</ol>
