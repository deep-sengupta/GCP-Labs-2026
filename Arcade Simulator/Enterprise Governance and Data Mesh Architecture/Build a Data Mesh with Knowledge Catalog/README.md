# Build a Data Mesh with Knowledge Catalog: Challenge Lab

<h2>📋 Steps</h2>

<ol>
  <li>Open <b>Cloud Shell</b>.</li>

  <li>Run the following commands:</li>

```bash
curl -LO
sudo chmod +x GSP514.sh
./GSP514.sh
```

  <li>Open <b>Data Catalog → Metadata Types</b>.</li>

  <li>Click <b>Create</b>.</li>

  <li>Enter the required <b>Display Name</b> and <b>Region</b>.</li>

  <li>Add the first field by entering its <b>Display Name</b>, <b>Type</b>, and <b>Values</b>.</li>

  <li>Add another field and enter the second <b>Display Name</b>, <b>Type</b>, and <b>Values</b>.</li>

  <li>Click <b>Save</b>.</li>

  <li>Search for <b>Raw Customer Zone</b> and open <b>raw_customer_zone</b>.</li>

  <li>Click <b>Add Optional Aspect</b> and then click <b>Save</b>.</li>

  <li>Return to <b>Cloud Shell</b>, press <b>Enter</b>, and provide the <b>User 2 email address</b>.</li>

  <li>Open <b>Data Profiling &amp; Quality</b>.</li>

  <li>Select the <b>Display Name</b>.</li>

  <li>Click <b>Run Now</b>.</li>
  
</ol>
